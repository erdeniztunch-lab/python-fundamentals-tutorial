## üõ†Ô∏è Troubleshooting

### Common Issues

**SyntaxError**
```
Problem: Missing colon, incorrect indentation
Solution: Check your colons after if/for/def and ensure proper indentation
```

**IndentationError**
```
Problem: Inconsistent spacing
Solution: Use 4 spaces for each indentation level
```

**NameError**
```
Problem: Using undefined variable
Solution: Make sure variables are defined before use
```

**TypeError**
```
Problem: Wrong data type for operation
Solution: Check data types and use appropriate conversions
```
